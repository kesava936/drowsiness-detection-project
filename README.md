# DrowsiGuard: Real-time AI Driver Drowsiness Detection

Hi! I built this web-based computer vision application to help prevent road accidents. It works by detecting driver fatigue through facial landmark analysis in real-time.

## Key Features I Implemented
* **Eye Aspect Ratio (EAR) Monitoring**: It constantly tracks the eyes to detect prolonged closure.
* **Yawn Detection**: I added tracking for mouth movements to spot signs of exhaustion.
* **Instant Audio Alerts**: When the app detects drowsiness, it triggers real-time warnings to wake up the driver.
* **Web-First & Privacy-Focused**: I designed this so it requires no installation. It runs entirely in the browser via your webcam, meaning no video data is sent to a server.

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Allow camera permissions and start monitoring!
